The Windows version were not extensively tested and therefore potentially subjected to more bugs and problems than the Mac ones.For bugreports, feature-requests and success-stories please contact us here: http://www.icst.net/about-the-icst/contact/These externals can be downloaded from www.icst.net/downloads----------------------------Installation instructions: (for 2.0) 2011/03/24     OS X (Max/MSP 5.1.7)   - place ICST-ambisonics folder into "/Cycling '74/" folder, ideally into a dedicated location for 3rd party externals.- in order to use the ambimonitor from the GUI-palette, move the .svg and .json files from "move-into-c74-object-palettes" to the "/Cycling '74/object-palettes" folder ----------------------------The ICST Ambisonics Tools have been developed at the Institute for Computer Music and Sound Technology of the Zurich University of the Arts by:- Jan Schacher- Philippe Kocher----------------------------
